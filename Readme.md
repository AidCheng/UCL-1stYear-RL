  - Beginning Java Programming, *Bart Baesens*
- Theory of Computation

  - Algorithms, *Robert Sedgewick*
  - Algorithms in C++, *Robert Sedgewick*
  - Discrete mathematics for computer scientist, *Truss, J.K.*
